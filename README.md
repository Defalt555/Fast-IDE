# Fast-IDE

Fast is an IDE that respect one major aspect : Speed

It should be a blazingly fast IDE with hotkeys to allow a faster writing speed.

### Frontend :

JS and Electron framework

### Backend :

C++ (for now)

### The entire project consist of :

- A text editor (LSP...)
- A project manager
- A compiler/interpreter integration
- A debugger integration (meaning you can place breakpoints, examine watchlist, evaluate expression at runtime, etc.)
